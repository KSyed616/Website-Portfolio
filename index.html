Projects

Finite State Machine

The project was coded using C in a virtual Linux environment. The finite state machine required the creation of a specific state machine that was provided, along with proper coding of the various relationships between each state. There was also options to remove "garbage" states i.e. ones with no relationships, change the relationships present and display the full machine. The process to make this machine required precise planning as the machine itself had 8 states, and proper understanding of how a state machine function.

General Purpose processor

The project was create in Quartus and coded with VHDL. The project consisted of 2 parts, the construction of processor and it's testing using ALUS. The processor istelf required various components including latches, 4-16 decoder and a Mealy Finite State Machine. The ALU portion consisted of 3 different ALUs to test the processors funcionality, the ALUs varied in difficulty, with some microcode functions being as easy AND of the two values and others involving shifting of the bits fully. The results of the processor were plotted on waveforms.
